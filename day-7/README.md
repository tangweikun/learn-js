## day-6

### Question

现有一数组 arr，数组里的元素全为整数；还有一整数 N；
如果 arr[i]小于 N，则将 arr[i]替换成 N，返回替换后的新数组

### Example1

```js
Input: [1, 2, 3, 4, 5], 2;
Output: [2, 2, 3, 4, 5];
```

### Example2

```js
Input: [3, 6, 3], 4;
Output: [4, 6, 4];
```

### Example3

```js
Input: [5, 6, 3], 2;
Output: [5, 6, 3];
```

### Example4

```js
Input: [], 2;
Output: [];
```
