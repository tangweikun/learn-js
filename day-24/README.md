## day-24

### Question

给定由字母组成的字符串 str，检查其是否包含连续重复的字母（a-zA-Z），包含返回 true，否则返回 false

### 知识点

- replace

### Example1

```js
Input: 'aad';
Output: true;
```

### Example2

```js
Input: '';
Output: false;
```

### Example3

```js
Input: 'ad';
Output: false;
```

### Example4

```js
Input: 'adad';
Output: false;
```

### Example5

```js
Input: 'adadcc';
Output: true;
```
