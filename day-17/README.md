## day-9

### Question

判断数组中是否所有的数字都大于 0 ，如果是就返回 true，否则返回 false，空数组也返回 false

### Example1

```js
Input: [1, 2, 3, 4, 5];
Output: true;
```

### Example2

```js
Input: [3, 6, -3];
Output: false;
```

### Example3

```js
Input: [5, 6, 0];
Output: false;
```

### Example4

```js
Input: [];
Output: false;
```
