## day-18

### Question

数组 arr 由正整数组成且按升序排列，求该数组中连续整数的最大个数

### 知识点

- Math.max

### Example1

```js
Input: [1, 2, 3, 8, 10];
Output: 3;
```

### Example2

```js
Input: [1, 6, 8];
Output: 1;
```

### Example3

```js
Input: [1, 2, 3, 8, 11, 12, 13, 14];
Output: 4;
```

### Example4

```js
Input: [];
Output: 0;
```
