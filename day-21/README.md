## day-21

### Question

已知 money 是一个数字，将其按如下规则转换

### 知识点

- toLocaleString
- ``

### Example1

```js
Input: 23442;
Output: '￥23,442';
```

### Example2

```js
Input: 123;
Output: '￥123';
```

### Example3

```js
Input: 123.34;
Output: '￥123.34';
```

### Example4

```js
Input: 12345679.87;
Output: '￥12,345,679.87';
```
