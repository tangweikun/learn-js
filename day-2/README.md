## day-2

### Question

现有一数组 arr，数组里的元素全是整数，请写一个方法返回不能被 3 整除的元素

### 相关知识点

>     push: 将一个元素添加到数组末尾

>     %: 取模操作

### Example1

```js
Input: [1, 3, 5, 9, 7];
Output: [1, 5, 7];
```

### Example2

```js
Input: [11, 2, 5];
Output: [11, 2, 5];
```

### Example3

```js
Input: [3, 6, 12];
Output: [];
```
