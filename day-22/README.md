## day-22

### Question

数组扁平化

### 知识点

- concat
- Array.isArray

### Example1

```js
Input: [1, 2, 3];
Output: [1, 2, 3];
```

### Example2

```js
Input: [1, 2, 3, [4, 5]];
Output: [1, 2, 3, 4, 5];
```

### Example3

```js
Input: [1, 2, 3, [4, 5], [[6], 7], []];
Output: [1, 2, 3, 4, 5, 6, 7];
```
