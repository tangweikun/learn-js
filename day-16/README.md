## day-16

### Question

如果数组全是偶数则返回 'EVEN', 如果全是奇数则返回'ODD', 如果奇数偶数都有就返回 'MIX'，否则返回'EMPTY'

### 知识点

> every

### Example1

```js
Input: [1, 2, 3, 4, 5];
Output: 'MIX';
```

### Example2

```js
Input: [3, 1, -3];
Output: 'ODD';
```

### Example3

```js
Input: [6, 0];
Output: 'EVEN';
```

### Example4

```js
Input: [];
Output: 'EMPTY';
```
